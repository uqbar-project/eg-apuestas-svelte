# Empezá tu propia svelte app

Este proyecto se comenzó a partir del template que provee [Svelte](https://svelte.dev)

Para crear un nuevo proyecto basándose en el mismo template usando [degit](https://github.com/Rich-Harris/degit):

```bash
npx degit sveltejs/template svelte-app
cd svelte-app
```

*Tenés que tener [Node.js](https://nodejs.org) instalado!*


Instalá las dependencias...

```bash
cd svelte-app
npm install
```

...lanzá la applicación con [Rollup](https://rollupjs.org):

```bash
npm run dev
```

Ya podés navegar a [localhost:5000](http://localhost:5000) y deberías ver tu aplicación corriendo. Editá los componentes dentro de `src`, guardá y verás los cambios en el browser.
